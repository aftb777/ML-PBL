# 📦 Demand Forecasting System using Machine Learning

A Machine Learning based Demand Forecasting System that predicts future product demand using historical sales data.
This project helps businesses avoid **overstocking**, **understocking**, and improves **inventory planning**.

---

## 🚀 Features

* Predict future product demand using ML
* Custom Product Dataset with Product ID
* Time-series based forecasting
* Simple Web Interface (HTML + CSS + Flask)
* Real-time demand prediction
* Lightweight and easy to run
* Beginner-friendly ML project

---

## 🧠 Tech Stack

* **Backend:** Python, Flask
* **Machine Learning:** Scikit-learn (Linear Regression)
* **Frontend:** HTML, CSS
* **Libraries:** NumPy, Pandas, Matplotlib (optional)

---

## 📊 How It Works

1. User enters **Product ID**
2. System retrieves historical sales of that product
3. Machine Learning model analyzes sales trend
4. Forecasts **next month demand**
5. Displays predicted demand on UI

## ▶️ Run Locally

### 1. Clone Repo

```bash
git clone https://github.com/aftb777/ML-PBL.git
cd ML-PBL
```

### 2. Create Virtual Environment

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install flask numpy scikit-learn
```

### 4. Run App

```bash
python app.py
```

Open in browser:

```
http://127.0.0.1:5000
```
